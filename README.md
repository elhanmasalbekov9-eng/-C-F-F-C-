#Конвертер температуры

## Описание проекта
Проект на Python для перевода температуры между шкалами:
- **Цельсий → Фаренгейт** (`c_to_f.py`)
- **Фаренгейт → Цельсий** (`f_to_c.py`)

Каждая программа:
1. Просит пользователя ввести температуру
2. Вычисляет результат по формуле
3. Показывает результат на экране


## Файлы проекта
- `c_to_f.py` — перевод из Цельсия в Фаренгейт
- `f_to_c.py` — перевод из Фаренгейта в Цельсий
