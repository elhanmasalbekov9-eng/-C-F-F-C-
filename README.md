# Конвертер температуры

## Описание проекта
Проект на Python для перевода температуры между шкалами:
- **Цельсий → Фаренгейт** (`c_to_f.py`)
- **Фаренгейт → Цельсий** (`f_to_c.py`)

Каждая программа использует функцию:
- `celsius_to_fahrenheit(c)` — возвращает температуру в Фаренгейтах
- `fahrenheit_to_celsius(f)` — возвращает температуру в Цельсиях

Каждая программа:
1. Просит пользователя ввести температуру
2. Вызывает функцию для вычисления
3. Вычисляет результат по формуле
4. Показывает результат на экране

## Файлы проекта
- `c_to_f.py` — перевод из Цельсия в Фаренгейт
- `f_to_c.py` — перевод из Фаренгейта в Цельсий
